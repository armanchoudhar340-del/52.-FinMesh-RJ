## 📖 About FinMesh

FinMesh is a modern digital asset management and settlement platform built using **React + Vite**. The system simulates institutional cryptocurrency operations, allowing users to track wallets, process settlements, monitor market spreads, analyze trading activity, and manage transaction history using core Data Structures & Algorithms.

The platform combines a futuristic fintech UI with practical DSA concepts such as Stacks, Queues, Hash Maps, Sorting Algorithms, and Search Algorithms.

## ✨ Features

### 💰 Wallet Balance Tracker
Track digital assets across multiple wallets and monitor portfolio performance.

- Multi-wallet support
- Asset distribution tracking
- Search functionality
- Portfolio valuation

### ↩️ Transaction Undo Log
Recover transactions using a LIFO-based rollback system.

- Stack implementation
- Transaction recovery
- Operation history
- Instant rollback

### 🔄 Settlement Queue
Manage trade settlements using FIFO processing.

- Queue implementation
- Settlement monitoring
- Pending transaction management
- Efficient execution flow

### 🆔 Account ID Checker
Verify and validate wallet identities.

- Account validation
- Ownership mapping
- Duplicate detection
- Hash-based lookup

### ⚡ Trade Sorter
Organize transactions efficiently.

- Sort by volume

- Sort by timestamp
- Optimized retrieval

### 📊 Price Difference Hub

- Spread analysis

### 📈 Analytics Dashboard
Gain insights into asset performance and network activity.

- Portfolio analytics

- Liquidity analysis
- Volume tracking

### 📦 Transaction Data Packer
Store and manage transaction records efficiently.

- Structured storage

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/armanchoudhar340-del/52-fin-mesh-rj.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
